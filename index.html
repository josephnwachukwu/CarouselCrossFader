This is a Carousel Plugin that allows you to scroll through not just pictures but anything!

You only need to perform two steps to begin using it. first include the CarouselCrossFade.js file somewhere on your page. or inside another js file. 

<script type="text/javascript" src="CarouselCrossFade.js"></script>

then call the tooltip funciton from your desired selector. 

$(document).ready(function() {
	$('.slideshow').carouselCrossFade();
})

right now there are there options that you can change.

  • the border of the slideshow
  • interval between slides
To do that you just add the attributes like this. 

$(document).ready(function() {
	$('.slideshow').carouselCrossFade({'delay' : '3000', 'border' : '#000 2px solid'});
})

In the next verson you will be able to control things like animation speed, the type of animation. and many more options.
